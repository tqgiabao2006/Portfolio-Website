Dissolve Effect

1. Maps the alpha value using a noise texture.

2. Increases the alpha clip threshold over time.

3. The VFX Graph spawns fire emblems.

4. Spawns animated butterflies.

Watch the Making Process Video

Portal Effect

1. Twist the main texture using polar coordinates.

2. Spin the UVs over time.

3. Use multiple meshes to create the portal effect.

Watch the Making Process Video

Water Effect

1. Calculates scene depth to change the color of the water based on depth.

2. Maps a noise texture and moves the UVs over time.

3. Calculates scene depth and surrounding objects; if they are equal, foam is created around them.

Watch the Making Process Video

Moving pixel effect

1. Saturates the UVs and divides by a constant resolution rate.

2. Uses Voronoi noise for mapping and moves it diagonally.

Watch the Making Process Video

Gold effect

1. Uses a normal map and metallic map.

2. Calculates the view direction with the dot product to separate the specular area.

3. Uses the Fresnel effect to create a glowing trim effect.

4. Optional toon shading.

Watch the Making Process Video
